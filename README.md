команды для запуска всего этого кода и что вообще делать (страшилка)


1. Запустить терминал
2. npm init -y (инициализация, всегда нужно что бы работали остальные команды)
3. npm i и npm i express pg postgres cors dotenv (установка нужных расширений, библиотек для работы приложения)

4. создаем базу данных через pgAdmin4 (можно вручную, можно взять код из файла tables.sql - рекомендуется)
5. редактируем файл по пути scripts/db.js (потербуется редактировать название базы данных (если нужно), а так же пароль для неё)
--(в редких случаях так же потребуется изменить порт на 5433 или 5431 или 8000 или 8888)

6. в файле package.json изменить "type": "common" на "type": "module" (если "type": "module" уже есть, то менять ничего не нужно)

6. node app.js или npm start (команда для запуска нашего сервера)

7. при помощи Live Server открыть главную страницу по пути public/index.html 
(правой кнопкой по index.html -> открыть с Live Server/ Open with Live Server)

ВАЖНО: команды для очистки таблиц (иногда, в случае ошибок с вставкой данных в таблицы лучше попробовать их очистить, и попробовать вставить данные снова)
:
TRUNCATE TABLE название_таблицы_для_очистки CASCADE;

(ЗАПУСКАТЬ ПО ОДНОЙ (можно и все три разом запустить, главное не запутайся что очистил, а что ещё не очищал))
TRUNCATE TABLE product_sales CASCADE;
TRUNCATE TABLE partner_types CASCADE;
TRUNCATE TABLE partners CASCADE;

команды для вывода данных из таблиц :
SELECT * FROM название_таблицы_для_вывода_данных;

(ЗАПУСКАТЬ ПО ОДНОЙ (сами по себе эти команды ничего не ломают, но если запустить сразу все три, то на экран будет выводиться результат последней команды))
SELECT * FROM product_sales;
SELECT * FROM partner_types;
SELECT * FROM partners;

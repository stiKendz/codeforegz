команды для запуска всего этого кода и что вообще делать (страшилка)

1. что бы перенести к себе репрезиторий:
git clone https://github.com/stiKendz/codeforegz (выполнять в терминале)
ИЛИ
тыкнуть на зеленую кнопку и выбрать download zip (потом открыть архив в VSCode)

2. переход в папку с проектом:
выполняем в терминале команду: cd codeforegz

!!! все дальшейшие команды так же нужно выполнять в терминале !!!

3. npm init -y (инициализация, всегда нужно что бы работали остальные команды)
4. npm i (установка нужных расширений, библиотек для работы приложения)
5. npm i express pg postgres cors dotenv (установка нужных расширений, библиотек для работы приложения)

6. создаем базу данных через pgAdmin4
7. создаем таблицы в базе данных (можно вручную, можно взять код из файла tables.sql - рекомендуется)

8. редактируем файл .env (не в pgAdmin4, в VSCode) --- (потербуется редактировать название базы данных (если нужно), а так же пароль для неё)
-- (в редких случаях так же потребуется изменить порт (поле DB_PORT) на 5433 или 5431 или 8000 или 8888)
-- аналогично для поля PORT - если 3000 не фурычит, то можно 8001 или 5000 или 5001, но это редкий случай что 3000 может не фурычить

9. в файле package.json изменить "type": "common" на "type": "module" (если "type": "module" уже есть, то менять ничего не нужно)
!!! "type": "git", игнорируем !!!

10. node server.js или npm start (команда для запуска нашего сервера)
11. тыкаемся по этой ссылке --- 🌐 Главная страница: http://localhost:3000 (важно - скрипт в терминале должен быть запущен, иначе откроет пустую страницу, и напишет что она не доступна (прекратить выполнение скрипта в терминале - ctrl + c))



(ВСЕ ЭТИ КОМАНДЫ МОЖНО НАЙТИ В ФАЙЛИКЕ selectTruncateTables.sql)
ВАЖНО: команды для очистки таблиц (иногда, в случае ошибок с вставкой данных в таблицы лучше попробовать их очистить, и попробовать вставить данные снова)
:
TRUNCATE TABLE название_таблицы_для_очистки CASCADE;

(ЗАПУСКАТЬ ПО ОДНОЙ (можно и все три разом запустить, главное не запутайся что очистил, а что ещё не очищал))
TRUNCATE TABLE product_sales CASCADE;
TRUNCATE TABLE partner_types CASCADE;
TRUNCATE TABLE partners CASCADE;

команды для вывода данных из таблиц :
SELECT * FROM название_таблицы_для_вывода_данных;

(ЗАПУСКАТЬ ПО ОДНОЙ (сами по себе эти команды ничего не ломают, но если запустить сразу все три, то на экран будет выводиться результат последней команды))
SELECT * FROM product_sales;
SELECT * FROM partner_types;
SELECT * FROM partners;
